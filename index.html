Interactive Preview

Welcome to Empathy Lens. This tool helps you experience how people with different abilities interact with digital content.

By toggling the options on the left, you can simulate various accessibility challenges:

  • Dyslexia simulation creates a jumbled text experience to help understand reading difficulties.
  • Color perception filters demonstrate how people with color vision deficiency see your content.
  • Motor control simulation demonstrates the challenges faced by users with limited motor skills.
  • Text-to-speech helps understand different comprehension speeds.

About This Project

Empathy Lens helps developers understand accessibility challenges by simulating different conditions. This tool promotes inclusive design principles.

Built entirely with HTML, CSS, and JavaScript - no frameworks or libraries. Demonstrates modern browser APIs to create meaningful experiences.

How to Use

Toggle the switches in the control panel to activate different simulations. Combine them to experience how multiple accessibility challenges might interact.

Experiment with the intensity sliders to adjust the simulation strength. Try to navigate and read content with different settings enabled.
